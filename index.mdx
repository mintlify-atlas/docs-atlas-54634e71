Bodega de Frío is a multi-role cold storage management system built with Next.js and Firebase. It gives your team a real-time view of every warehouse slot, tracks inbound and outbound box flows, monitors temperatures, and triggers alerts automatically — all synchronized across sessions through Firestore.

## What Bodega de Frío does

<Steps>
  <Step title="Manage warehouses and slots">
    Create multiple warehouses, each with a configurable number of cold storage slots. Each slot tracks which box occupies it, the stored item's name, temperature, and client assignment.
  </Step>
  <Step title="Track inbound, storage, and outbound flows">
    Boxes move through three zones — **Ingresos** (inbound), **Bodega** (storage), and **Salida** (outbound). Operators and supervisors issue orders to move boxes between zones, and the system queues pending work in real time.
  </Step>
  <Step title="Monitor temperatures and respond to alerts">
    Any box whose temperature exceeds 5 °C triggers an automatic alert. Supervisors assign alerts to operators, who resolve them and log the outcome. Overdue orders (pending more than 2 minutes) also raise alerts automatically.
  </Step>
  <Step title="Generate reports and audit history">
    Every ingreso, salida, and bodega movement is recorded in a history log. The Reportes section lets authorized users export activity summaries as PDF for compliance and auditing.
  </Step>
</Steps>

## Key capabilities

  <Card title="Multi-warehouse support" icon="warehouse">
    Switch between internal and external warehouses (including Fridem integration) from a single dashboard.
  </Card>
  <Card title="Role-based access" icon="shield-halved">
    Six distinct roles — custodio, administrador, operario, jefe, cliente, and configurador — control what each user can see and do.
  </Card>
  <Card title="Real-time sync" icon="rotate">
    All warehouse state is synchronized via Firestore listeners, so every connected session sees the same data instantly.
  </Card>
  <Card title="Temperature alerts" icon="temperature-high">
    Automatic alerts fire when any box exceeds the 5 °C threshold, with assignment and resolution tracking.
  </Card>
  <Card title="Order queue" icon="list-check">
    Supervisors issue move and review orders; operators see their task queue live and complete orders step by step.
  </Card>
  <Card title="PDF reports" icon="file-pdf">
    Export activity history and warehouse state summaries as PDF directly from the Reportes tab.
  </Card>
